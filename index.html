Unreal Engine 4 RTS Kit

An Unreal Engine 4 plugin for creating RTS games

Features:

  • 100% C++
  • 100% multiplyer supported
  • Teams
  • Factions
  • Fog of war
  • CPU players
  • Match Observers
  • Several different builing build methods (C&C RA3 Allies and Soviets, C&C Generals style)
  • Defense structures
  • Garrisonable buildings
  • Upgrades
  • Building/unit abilities
  • Buffs/debuffs
  • Inventories and items
  • "Mana"
  • Damage types and armor types
  • Targeting types (e.g. missle turrets cannot attack ground units)
  • Stealth units
  • Resources (e.g. minerals, vespene gas, ore)
  • Unit experience and ranking up
  • Building/unit/upgrade prerequisites (e.g. build a power plant before you can build a barracks)
  • Unit uniqueness (e.g. you can only have 1 tanya at a time)
  • Player experience and ranking up
  • Player abilities

  • UI fully implemented
  • In match chat and lobby chat
  • Healthbars, manabars
  • Main menu, match lobby, all settings menus (video, audio, controls, game, key bindings), HUD
  • Marquee unit selection
  • "Control" groups unit selection (e.g. press 1 to select some units)
  • Camera movement due to mouse at edge of screen
  • Hardware cursors
